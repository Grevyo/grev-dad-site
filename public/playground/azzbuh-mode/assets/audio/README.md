# Azzbuh Mode Audio Dropzone

Put fart sound files in this folder so the **Pull my Digital Finger** button can play random clips.

Expected filenames:
- `fart-1.mp3`
- `fart-2.mp3`
- `fart-3.mp3`
- `fart-4.mp3`

You can add more files later by extending the `fartSoundPaths` array in:
`/public/playground/azzbuh-mode/script.js`
