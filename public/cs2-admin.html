Manual catalog workflow

The simulator seeds a researched built-in catalog for the requested CS2 cases, and nothing auto-syncs after that. Upload your own text file if you want to wipe and replace the catalog, then use the buttons below whenever you want to refresh saved Steam prices or images.

Upload case-opener text file

Upload the fixed list of skins you want available in the case opener. Supported formats: Case: Name headings followed by Weapon | Skin | Rarity, or one row per line like Case | Weapon | Skin | Rarity | Wear | Market Hash. During import, the admin tool will fetch and save the image plus one stored market price for each case and skin.

Quick sell fee

Players receive (100 − fee)% of the live market reference when quick selling. Default 10% fee → 90% payout.

Quick-sell graveyard

Admins can see quick-sold tracked skins and refund them back to players if a sell was accidental.

Image reports

Players can flag bad skin or case artwork here. Resolve one image at a time or bulk refresh all reported entries.
